# Corn Production Analysis in Indonesia

## Background
## Tools

The tools used in this project are a follows:

- Python Notebook
- Google Locker Studio
  
## Process

The Process in this project includes:

- Combine all dataset into a single dataset
- Identify the data structure such as missing value, duplicate data, type of data and create a data summary
- Perform preprocessing to improve the dataset, such as changing column data types, handling missing values and adding new columns based on available information
- Create data visualization using Notebook and compile dashboard using Google Locker Studio
