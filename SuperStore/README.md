## Tools

The tools used in this project are a follows:
- Python Notebook
- Google Locker Studio


## Process

The process of this project includes:
- Identify the dataset structure, including missing values and duplicate data and create a data summary
- Perform Preprocessing to improve the dataset
- Create Data Visualization using Notebook (IPYNB) and compile dashboard using Google Locker Studioo
